📒 Sistema de Cadastro de Contatos em Python
📌 Descrição

Este é um sistema simples de cadastro de contatos desenvolvido em Python.
O projeto permite adicionar, listar e armazenar contatos utilizando arquivos JSON para persistência de dados.

🚀 Funcionalidades
✔ Adicionar contatos com nome e idade
✔ Listar contatos cadastrados
✔ Persistência de dados em arquivo JSON
✔ Verificação de contatos duplicados
✔ Interface simples via terminal (CLI)

🧠 Conceitos aplicados
Programação Orientada a Objetos (POO)
Manipulação de arquivos JSON
Estruturas de repetição (while True)
Condicionais (if / elif / else)
Funções auxiliares
Validação de dados
Entrada de dados via terminal

⚙️ Como executar o projeto
# Clone o repositório
git clone https://github.com/AxelOliveira/person_json_registration

# Acesse a pasta
cd person_json_registration

# Execute o programa
python main.py

🖥️ Como usar

Ao iniciar o sistema, você verá os comandos disponíveis:

Comandos: adicionar contato, listar contatos ou sair.
